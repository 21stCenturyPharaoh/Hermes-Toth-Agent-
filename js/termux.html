Zero SMS. Zero Cost. 7-Color Terminal Alerts via `hermes.sh` polling.

#!/data/data/com.termux/files/usr/bin/sh
# hermes.sh | Polls .hermes_pings.sh queue
while true; do
  if [ -f .hermes_pings.sh ]; then source .hermes_pings.sh; fi
  sleep 10
done

See `js/termux.js` for ping formats.
